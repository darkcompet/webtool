# Log for dotnet

This provides logging for dotnet.

- Features

	```bash
	- Logging.
	```

- Dependencies

	```bash
	# Tool.Compet.Core
	https://github.com/darkcompet/csharp-core
	```


## How to install

- Just include it into project by add as git-submodule.
	
	```bash
	# Tool.Compet.Log
	git submodule add https://github.com/darkcompet/csharp-net-log.git
	```
